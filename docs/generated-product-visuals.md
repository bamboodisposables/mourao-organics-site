# Generated Product Visuals

Mourão Organics product imagery is now generated locally from code.

- Builder: [scripts/build-product-assets.py](/Users/OlivierJacobs_1/Desktop/Mourao/scripts/build-product-assets.py)
- Output: `assets/mourao-product-*-hero.jpg`, `assets/mourao-product-*-card.jpg`, `assets/mourao-product-*-thumb-*.jpg`
- Inputs: brand logo asset only
- External product photography: none

The renderer creates:

- branded jars, tubes, pump bottles and tins
- warm beige studio backgrounds
- soft shadows, highlights and ingredient props
- multiple thumb variants per product
